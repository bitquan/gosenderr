# Xcode readiness checklist

Use this checklist before building in Xcode.

## Required
- [ ] Xcode installed and opened at least once
- [ ] Node.js 18+
- [ ] pnpm installed
- [ ] Ruby + Bundler installed
- [ ] CocoaPods available
- [ ] pnpm install completed at repo root
- [ ] bundle install completed in ios/
- [ ] bundle exec pod install completed in ios/

## App configuration
- [ ] Firebase config valid
- [ ] Mapbox token available

## Build steps
- [ ] Open ios/Senderrappios.xcworkspace
- [ ] Select Senderrappios scheme
- [ ] Build and run on a Simulator
