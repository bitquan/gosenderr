# Admin Desktop Branching Template

Use `admin-desktop/*` branches for Admin Desktop scoped work.

## Branch naming

- `admin-desktop/feature/<short-name>`
- `admin-desktop/fix/<short-name>`
- `admin-desktop/docs/<short-name>`

## Scope guard

- Keep changes in:
  - `apps/admin-desktop`
  - `docs/admin_desktop`

## V2 policy

- Do not introduce new Senderrplace V2 admin dependencies here while Admin Web is canonical.
