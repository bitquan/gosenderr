# SHORTCUTS.md

Keyboard shortcuts for the Admin Desktop app (Phase 1).

Navigation:
- Cmd/Ctrl+1: Dashboard
- Cmd/Ctrl+2: Users
- Cmd/Ctrl+3: Orders
- Cmd/Ctrl+4: Jobs
- Cmd/Ctrl+5: Disputes

Common actions:
- Cmd/Ctrl+F: Search
- Cmd/Ctrl+R: Refresh
- Cmd/Ctrl+, : Settings
- Cmd/Ctrl+Q: Quit

Window:
- Cmd/Ctrl+W: Close window
- Cmd/Ctrl+M: Minimize
- Cmd/Ctrl+Plus: Zoom in
- Cmd/Ctrl+Minus: Zoom out

Add new shortcuts to `apps/admin-desktop/electron/menu.ts` and document any platform-specific differences here.
