# Senderr Web Branching Template

Use `senderr-web/*` branches for Senderr Web scoped work.

## Branch naming

- `senderr-web/feature/<short-name>`
- `senderr-web/fix/<short-name>`
- `senderr-web/docs/<short-name>`

## PR target

- Base: `senderr_app` unless explicitly directed otherwise.

## Scope guard

- Keep changes in:
  - `apps/senderr-app`
  - `docs/senderr_web`
  - shared files only when required.
