# Admin Web Branching Template

Use `admin-app/*` branches for Admin Web scoped work.

## Branch naming

- `admin-app/feature/<short-name>`
- `admin-app/fix/<short-name>`
- `admin-app/docs/<short-name>`

## Scope guard

- Keep changes in:
  - `apps/admin-app`
  - `docs/admin_app`

## V2 policy

- New Senderrplace V2 admin controls ship in Admin Web first.
