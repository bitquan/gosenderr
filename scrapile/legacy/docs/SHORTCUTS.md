# SHORTCUTS.md

Keyboard shortcuts for the Admin Desktop app (Phase 1).

Navigation:
- Cmd/Ctrl+1: Dashboard
- Cmd/Ctrl+2: Users
- Cmd/Ctrl+3: Marketplace Orders
- Cmd/Ctrl+4: Jobs
- Cmd/Ctrl+5: Disputes

Common actions:
- Cmd/Ctrl+K: Open Global Search
- Esc: Close Global Search

Window:
- Cmd/Ctrl+Q: Quit (system default)

Add new shortcuts to `apps/admin-desktop/electron/menu.ts` and document any platform-specific differences here.
